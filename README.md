# Dashboard SuperStore Sales

Esse projeto se trata da transformação do arquivo csv em que está contido o **famoso dataset do Kaggle: "SuperStore Sales"** para um dashboard do **Power BI**.

## Como Usar

* Realize o **Fork** do Arquivo para sua máquina.
* Abra com um navegador ou app do **Power BI**.

## Aviso

* Por questão de redundância todo o dashboard está em **inglês**, visto que o dataset todo também está.

* Apenas a versão de desktop está disponível no momento, a versão mobile sairá em breve.
